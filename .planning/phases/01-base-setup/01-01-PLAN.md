<task type="auto">
  <name>Task 2: Start server and client, verify they boot</name>
  <files></files>
  <action>
1. Check ports are free:
   ```
   lsof -i :3000
   lsof -i :5173
   ```
   Kill any conflicting processes if needed.

2. Start the server in background:
   ```
   cd server && yarn dev
   ```
   Expected output: "Server started on port 3000, allowed cors origin: http://localhost:5173"
   The server uses nodemon for auto-restart.

3. Start the client in background:
   ```
   cd client && yarn dev
   ```
   Expected output: Vite dev server URL showing http://localhost:5173

4. Verify both processes are running:
   - `curl -s http://localhost:3000` should get a response (Socket.IO serves a page)
   - `curl -s http://localhost:5173` should return HTML

5. Check browser console would be clean by verifying no immediate crash in either process's terminal output. Look for:
   - No CORS errors in server output
   - No module resolution errors
   - No port conflict errors

IMPORTANT: Server must start BEFORE client tries to connect.
IMPORTANT: If port 3000 is in use, kill the process. Do not change the port (CORS is hardcoded).
  </action>
  <verify>
- Server process running on port 3000 (check with `lsof -i :3000`)
- Client process running on port 5173 (check with `lsof -i :5173`)
- `curl -s http://localhost:5173` returns HTML containing "root" div
- Neither process shows errors in its output
  </verify>
  <done>Server running on port 3000, client running on port 5173, both started without errors.</done>
</task>
