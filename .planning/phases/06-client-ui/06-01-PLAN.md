Inside the SocketManager component:

2. Store a ref (`charactersRef`) that tracks the latest `characters` value (update it in a useEffect whenever characters changes). This is needed because the socket event handler closure won't see updated atom values.
3. Add a `playerChatMessage` socket listener in the existing useEffect (alongside the other socket listeners):
   ```
   function onPlayerChatMessage(value) {
     // value = { id: socketId, message: string }
     const chars = charactersRef.current || [];
     const sender = chars.find(c => c.id === value.id);
     setChatMessages(prev => {
       const next = [...prev, {
         id: `${Date.now()}-${value.id}`,
         senderId: value.id,
         senderName: sender?.name || 'Player',
         isBot: sender?.isBot || false,
         message: value.message,
         timestamp: Date.now(),
       }];
       return next.slice(-20); // Keep last 20 messages
     });
   }
   ```
4. Register `socket.on("playerChatMessage", onPlayerChatMessage)` and clean up with `socket.off` in the return.
