3. Create `bot/BotClient.js` with:

   **Class structure:** `BotClient extends EventEmitter` from `node:events`

   **Constructor** accepts `{ serverUrl, avatarUrl, name }`:
   - `this.serverUrl` = serverUrl (default `"http://localhost:3000"`)
   - `this.avatarUrl` = avatarUrl (default `"https://models.readyplayer.me/64f0265b1db75f90dcfd9e2c.glb"`)
   - `this.name` = name (default `"ClawBot"`)
   - `this.socket` = null
   - `this.id` = null (socket.id from server)
   - `this.room` = null (room map data)
   - `this.position` = null (current grid [x, y])
   - `this.rooms` = [] (available rooms from welcome)
   - `this.characters` = [] (current room characters)

   **`connect()` method** — returns Promise:
   - Create socket with `io(this.serverUrl, { transports: ["websocket"], autoConnect: true, reconnection: true, reconnectionAttempts: 5, reconnectionDelay: 1000 })`
   - Use `transports: ["websocket"]` to bypass CORS (critical — see research Pitfall 3)
   - On `"welcome"` event: store `data.rooms` in `this.rooms`, emit `"welcome"` on self, resolve promise with data
   - On `"connect_error"`: reject promise with error
   - Forward socket events to BotClient EventEmitter:
     - `"characters"` → `this.characters = chars`, emit `"characters", chars`
     - `"playerMove"` → emit `"playerMove", character`
     - `"playerChatMessage"` → emit `"chatMessage", data` (data has {id, message})
     - `"emote:play"` → emit `"emote", data` (data has {id, emote})
     - `"playerDance"` → emit `"dance", data` (data has {id})
     - `"mapUpdate"` → update `this.room` and `this.characters`, find own position, emit `"mapUpdate", data`
     - `"disconnect"` → emit `"disconnected"`

   **`join(roomId)` method** — returns Promise:
   - If no socket or not connected, throw Error "Not connected"
   - Set timeout of 5000ms, reject with "Join timeout" if no response
   - Listen once for `"roomJoined"` event:
     - Clear timeout
     - Store `data.id` as `this.id`
     - Store `data.map` as `this.room`
     - Store `data.characters` as `this.characters`
     - Find own character: `data.characters.find(c => c.id === this.id)`
     - Store own character's `position` as `this.position`
     - Emit `"joined", data` on self
     - Resolve with data
   - Emit `"joinRoom"` with `(roomId, { avatarUrl: this.avatarUrl, isBot: true, name: this.name })`

   **`leave()` method:**
   - If no socket or no room, return silently
   - Emit `"leaveRoom"` on socket
   - Reset `this.room`, `this.position`, `this.id`, `this.characters` to null/[]

   **`disconnect()` method:**
   - If socket exists, call `this.socket.disconnect()`
   - Reset all state (room, position, id, rooms, characters)

   **`get connected()` getter:**
   - Returns `this.socket?.connected ?? false`

   Export `BotClient` as named export.
