Create `bot/BotBridge.js` that exports a `BotBridge` class. This is the main orchestrator.

**Constructor:** `new BotBridge(options)`

Options:
- `serverUrl` -- game server URL (default from `SERVER_URL` env or `http://localhost:3000`)
- `gatewayUrl` -- Gateway URL (default from `OPENCLAW_GATEWAY_URL` env or `ws://localhost:8080`)
- `gatewayToken` -- Gateway auth token (from `OPENCLAW_GATEWAY_TOKEN` env)
- `botName` -- bot display name (default "ClawBot")
- `avatarUrl` -- Ready Player Me avatar URL (default existing one from BotClient)
- `loopIntervalMs` -- baseline perception-decision loop interval (default 3000)
- `perceptionRadius` -- nearby filter radius (default 6)
- `rateLimitBurst` -- burst tokens (default 3)
- `rateLimitSustained` -- sustained tokens/sec (default 1)
- `debug` -- enable visible debug info (from `BOT_DEBUG=1` env, default false)

**Initialization (in constructor):**
- Create `this._botClient = new BotClient({ serverUrl, avatarUrl, name: botName })`
- Create `this._gateway = new GatewayClient({ url: gatewayUrl, token: gatewayToken })`
- Create `this._perception = new PerceptionModule(this._botClient, { radius: perceptionRadius })`
- Create `this._idle = new IdleController(this._botClient)`
- Create `this._rateLimiter = createRateLimiter({ burst: rateLimitBurst, sustained: rateLimitSustained })`
- Create `this._log = createBotLogger(null, botName)` (botId set later after join)
- State: `this._state = "init"` (init -> spawning -> active -> stopping -> stopped)
- `this._loopTimer = null`
- `this._gatewayConnected = false`
- `this._pendingDecision = false` (prevents overlapping LLM calls)

**Wire up event handlers in constructor:**

```js
// Chat messages -> perception module
// BotClient emits "chatMessage" with payload { id, message } (forwarded from server's
// "playerChatMessage" event). PerceptionModule.onChatMessage expects { id, message }.
// Explicitly map to ensure contract matches even if server payload changes:
this._botClient.on("chatMessage", (data) => {
  this._perception.onChatMessage({ id: data.id, message: data.message });
  // Immediate trigger: if someone talks and we're not already deciding, trigger loop early
  if (!this._pendingDecision && this._gatewayConnected) {
    this._triggerLoop();
  }
});

// Gateway connection status
this._gateway.on("connected", () => {
  this._gatewayConnected = true;
  this._log.info("Gateway connected");
});
this._gateway.on("disconnected", () => {
  this._gatewayConnected = false;
  this._log.warn("Gateway disconnected -- entering idle-only mode");
});
this._gateway.on("reconnecting", ({ attempt, delay }) => {
  this._log.info({ attempt, delay }, "Gateway reconnecting");
});
```

**`async start()` method -- spawn lifecycle:**
1. Set `_state = "spawning"`
2. Connect to Gateway: `await this._gateway.connect()`. Set `_gatewayConnected = true`.
3. Connect to game server: `const welcome = await this._botClient.connect()`
4. Join first available room: `const room = welcome.rooms[0]; await this._botClient.join(room.id)`
5. Update logger with botId: `this._log = createBotLogger(this._botClient.id, botName)`
6. Log: `{ room: room.name, position: this._botClient.position }` "Bot spawned"
7. Set `_state = "active"`
8. Start idle controller: `this._idle.start()`
9. Start main loop: `this._startLoop()`
10. Return `{ id: this._botClient.id, room: room.name }`

**`_startLoop()` method:**
Set `this._loopTimer = setInterval(() => this._tick(), loopIntervalMs)`.

**`_triggerLoop()` method:**
Cancel current timer, call `_tick()` immediately, then restart interval.

**`async _tick()` method -- the core perception-decision-action cycle:**

```
1. If _state !== "active" or _pendingDecision, return
2. Set _pendingDecision = true
3. const startMs = Date.now()
4. Try:
   a. Take perception snapshot: snap = this._perception.snapshot()
   b. If no nearby players and gateway connected:
      - Run idle tick: this._idle.tick()
      - Set _pendingDecision = false, return
   c. If gateway not connected:
      - Run idle tick: this._idle.tick()
      - Set _pendingDecision = false, return
   d. Serialize perception: text = this._perception.serialize(snap)
   e. Build prompt for LLM (see below)
   f. Interrupt idle (bot will do LLM-directed action): this._idle.interrupt()
   g. Call Gateway: result = await this._gateway.invokeAgent(prompt)
   h. Extract action text from result (handle both string and object -- if result has .text use that, if result has .content use that, if string use directly)
   i. Parse action: parsed = parseAction(actionText)
   j. If !parsed.ok:
      - Log warn: { error: parsed.error, raw: actionText }, "Invalid LLM action, falling back to idle"
      - Retry once: re-invoke with simpler "respond with valid JSON" prompt
      - If retry also fails: resume idle, set _pendingDecision = false, return
   k. Check rate limiter: if !this._rateLimiter.tryConsume():
      - Log info "Action rate-limited, queuing"
      - await this._rateLimiter.waitForToken()
   l. Execute action: execResult = executeAction(parsed.action, this._botClient, this._log)
   m. Record own action: this._perception.recordOwnAction(parsed.action)
   n. Log info: { action: parsed.action.type, latencyMs: Date.now() - startMs, cycle: "complete" }
5. Catch (err):
   - Log error: { err }, "Decision cycle failed"
   - Resume idle: this._idle.tick()
6. Finally:
   - Set _pendingDecision = false
```

**LLM Prompt construction:**

```js
const prompt = `You are ${botName}, a friendly bot in a multiplayer 3D room. Based on what you perceive, decide your next action.

${text}

Respond with exactly ONE JSON action. Available actions:
- {"type":"move","target":[x,y]} - Walk to grid position
- {"type":"say","message":"..."} - Say something (max 200 chars)
- {"type":"emote","name":"wave|dance|sit|nod"} - Perform emote
- {"type":"look","target":"playerName"} - Face a player

Rules:
- Be social and friendly. Greet nearby players.
- If someone spoke recently, respond to them.
- If alone, explore or emote.
- Respond ONLY with the JSON object, no explanation.`;
```

**`async stop()` method -- cleanup lifecycle:**
1. Set `_state = "stopping"`
2. Clear loop timer
3. Stop idle controller: `this._idle.stop()`
4. Destroy rate limiter: `this._rateLimiter.destroy()`
5. Bot client leave and disconnect: `this._botClient.leave(); this._botClient.disconnect()`
6. Gateway disconnect: `this._gateway.disconnect()`
7. Set `_state = "stopped"`
8. Log info "Bot stopped"

```js
import { BotBridge } from "./BotBridge.js";
import { createLogger } from "./logger.js";

const log = createLogger({ name: "bot-main" });

const bridge = new BotBridge({
  botName: process.env.BOT_NAME || "ClawBot",
});

// Graceful shutdown
const shutdown = async (signal) => {
  log.info({ signal }, "Shutting down...");
  await bridge.stop();
  log.info("Shutdown complete");
  process.exit(0);
};

process.on("SIGINT", () => shutdown("SIGINT"));
process.on("SIGTERM", () => shutdown("SIGTERM"));

try {
  const info = await bridge.start();
  log.info(info, "Bot is live");
} catch (err) {
  log.error({ err }, "Failed to start bot");
  process.exit(1);
}
```

Key behaviors:
- Reads BOT_NAME from env (default "ClawBot")
- All other config from env vars (SERVER_URL, OPENCLAW_GATEWAY_URL, OPENCLAW_GATEWAY_TOKEN)
- Graceful shutdown on SIGINT/SIGTERM calls bridge.stop()
- Top-level error handling exits with code 1
- Uses structured logger for startup/shutdown messages
